Hyperlinks

Hyperlink ou link é uma referência de um documento em hipertexto para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico.

Como usar as tags de marcação

Para fazer uma marcação de um hyperlink usamos a tag <a> e a propriedade href que faz referência ao conteúdo.

Um exemplo prático é você clicar nesse texto e ser mandado direto para a página inicial do Google.

Agora se você clicar novamente no texto você será redirecionado para uma nova aba.
