Software de Versionamento

É um sistema que tem por objetivo realizar o controle de versões de um código ou documento.
É possível gerenciar as versões de um código ou documento realizadas por um desenvolvedor ou equipe de desenvolvedores.

Repositório

Quando você escreve seu projeto no seu computador, você o armazena em uma pasta no seu computador, com o software de Versionamento, você o armazena online, ou seja, um repositório remoto.

Sempre que você fizer uma alteração no seu projeto, ele irá alterar no repositório também, criando uma nova versão.

Tipos de Software de Controle de Versão

Centralizado - Possuí um repositório central em um servidor de controle de versões. Depende do servidor, se o servidor cair, você não poderá acessá-lo.

Distribuido - Possuí um repositório local com controle de versões. A atualização no repositório central é independente do servidor. Você pode fazer suas alterações no computador e depois subir o projeto.
